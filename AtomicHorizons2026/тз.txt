Ядерные материалы

2. ХИМИЧЕСКАЯ ПЕРЕРАБОТКА: 5 ТИРОВ

Как в Mekanism: каждый тир умножает выход ресурсов, но требует всё больше инфраструктуры.

2.1 Тир 0: Примитивная обработка
Выход: 1 слиток = 1 руда

```
Урановая руда → Печь → Урановый слиток (низкая чистота, высокая радиация)
```

2.2 Тир 1: Механическое обогащение
Выход: 2 слитка = 1 руда

- Crusher → дробит руду в Crushed Ore
- Enrichment Chamber → обогащает до Enriched Ore Dust
- Smelter → переплавка в слитки

2.3 Тир 2: Кислотное выщелачивание
Выход: 3 слитка = 1 руда

- Chemical Dissolution Chamber → руда + H₂SO₄ = Radioactive Slurry
- Chemical Washer → промывка водой = Clean Slurry
- Chemical Crystallizer → кристаллизация = Uranium Crystals
- Smelter → слитки

2.4 Тир 3: Газовая диффузия
Выход: 4 слитка = 1 руда

- Chemical Oxidizer → кристаллы + кислород = Yellowcake (U₃O₈)
- Chemical Infuser → Yellowcake + HF = Uranium Hexafluoride (газ)
- Isotopic Centrifuge → разделение на U-235 (5%) и U-238 (95%)
- Chemical Injection Chamber → обратная конверсия в металл

2.5 Тир 4: Квантовая переработка
Выход: 5 слитка = 1 руда

- Laser Isotope Separator → лазерное разделение (без отходов!)
- Molecular Assembler → сборка на атомарном уровне
- Quantum Entanglement Chamber → мгновенная телепортация материалов между узлами

2.6 Машины химической линии

Dissolution Chamber (Камера растворения)
- Мультиблок 3x3x3
- Принимает: руда + кислота
- Выдает: шлам (slurry)
- Особенность: Если сломать во время работы — выливается кислота в мир!

Rotary Condensentrator
- Конвертирует жидкость ↔ газ
- Необходим для работы с UF₆

Solar Neutron Activator
- Превращает Lithium → Tritium при дневном свете
- Требует прямой видимости неба

Pressurized Reaction Chamber (PRC)
- 3 входа: предмет + жидкость + газ
- Высокое давление, риск взрыва при перегреве

4. БЕТОН И СТРОИТЕЛЬНЫЕ МАТЕРИАЛЫ

4.1 Компоненты бетона

Cement (Цемент)

```
Известняк + Глина + Песок → Cement Mixer → Cement Powder
Cement Powder + Вода → Cement Paste
```

Reinforced Concrete (Армированный бетон)

```
Cement Paste + Песок + Гравий + Rebar (арматура) → Liquid Concrete
```

Rebar (Арматура)

```
Steel Ingot → Rolling Machine → Steel Rod → 4x Rebar
```

4.2 Миксер (Concrete Mixer) — мультиблок

Размер: 3x3x4 (ширина x длина x высота)
Конструкция:
- Основание: 3x3 из Heavy Engineering Blocks
- Стены: Steel Scaffolding + Fluid Pipes
- Верх: Redstone Engineering Block (контроллер)

Процесс:
1. Загрузка компонентов в бункер
2. Заливка воды через трубу
3. Перемешивание (энергия: 512 RF/t)
4. Выход: Liquid Concrete в Fluid Tank или Concrete Bucket

4.3 Заливка бетона

Liquid Concrete можно:
- Вылить в мир (становится Wet Concrete)
- Wet Concrete застывает за 10 минут реального времени в Concrete
- Ускорение: применить Concrete Accelerator (хлористый кальций)

Свойства Concrete:
- Прочность: 50 (как обсидиан)
- Радиационная защита: блокирует 90% радиации
- Взрывостойкость: 100 (иммунитет к ядерным взрывам)

4.4 Специальные бетоны

Тип	Рецепт	Свойства	
Heavy Concrete	Concrete + Lead Dust	Защита 95%, вес x2	
Radiation Shielding Concrete	Concrete + Boron + Lead	Защита 99%, для реакторов	
Heat Resistant Concrete	Concrete + Кремнезем	Выдерживает 2000°C	
Structural Concrete	Concrete + Стальная опилка	Прочность x2, защита 80%

5. ГРАДИРНИ И ТЕПЛООБМЕН

5.1 Типы градирей

Passive Cooling Tower (Пассивная градирня)
- Высота: 4-16 блоков
- Форма: Гиперболоид (сужается кверху)
- Материал: Reinforced Concrete
- Принцип: Естественная тяга горячего воздуха

Mechanical Draft Tower (Механическая)
- Вентиляторы наверху (требуют энергию)
- Эффективность +40% vs пассивная
- Шум (звуковое загрязнение чанков)

Hybrid Cooling Tower (Гибридная)
- Сухое + мокрое охлаждение
- Минимальное испарение воды
- Дорогая в строительстве

5.2 Конструкция градирни (мультиблок)

Требования:
1. База: Круг диаметром (высота/2 + 2)
2. Стены: Reinforced Concrete, полые внутри
3. Внутренность: Fill Material (перекрестия для воды)
4. Верх: Открытый выход (иначе не работает!)
5. Вход: Горячая вода/пар снизу
6. Выход: Холодная вода в бассейн

Формула эффективности:

```
Efficiency = (Base 60%) + (Height Bonus: +2% per block above 8) + (Biome Bonus) - (Obstruction Penalty)
```

Biome Bonus:
- Пустыня: +15%
- Джунгли: +10%
- Тайга: -5%
- Ад: -50% (нет воды!)
- Край: -30%

5.3 Теплообменники

Shell-and-Tube Heat Exchanger
- Мультиблок 3x3x5
- Первичный контур (радиоактивный) ↔ Вторичный (чистый)
- Не допускает смешивания жидкостей!

Plate Heat Exchanger
- Компактный вариант
- Меньшая эффективность, но дешевле

Steam Generator
- Принимает: нагретый теплоноситель (натрий/свинец)
- Выдает: High-Pressure Steam для турбин

6.1 Типы радиации

Alpha	Радий, Плутоний	5 см воздуха	Бумага, кожа

6.2 Система накопления радиации

Chunk Radiation (Радиация чанка)
- Хранится в NBT данных чанка
- Распространяется в соседние чанки (1% в тик)
- Распадается со временем (полураспад: 1 час игрового времени)

Player Radiation (Доза игрока)
- Накапливается при нахождении в радиоактивной зоне
- Не уменьшается при выходе (только RadAway!)
- Эффекты по порогам:

Доза	Эффекты	
0-200 RAD	Нет	
200-400	Слабость I, тошнота (20% шанс)	
400-600	Слабость II, медлительность I, голод	
600-800	Слабость III, медлительность II, отравление	
800-1000	Иссушение, слепота, вероятность смерти	
1000+	Смерть, Achievement: "Marie Curie Award"	

Environmental Effects
- Grass → Dead Grass (при >100 RAD/чанк)
- Leaves → Withered Leaves
- Water → Contaminated Water (наносит радиацию при употреблении)
- Mobs → Мутации (Zombie Villagers, Nuclear Creepers)

6.3 Детекторы радиации

Geiger Counter (Счетчик Гейгера)
- Показывает радиацию чанка (mRAD/t)
- Звуковой сигнал: треск пропорционально дозе
- HUD индикация при экипировке

Dosimeter (Дозиметр)
- Показывает накопленную дозу игрока
- Можно сбросить только в Hospital Bed

Radiation Monitor Block
- Стационарный, подключается к Redstone
- Сигнал при превышении порога

Scintillation Detector
- Требует энергию
- Определяет ТИП радиации и источник

6.4 Деконтаминация

RadAway (Антирадин)
- Шприц, снижает дозу на 200 RAD
- Перезарядка: 5 минут
- Побочный эффект: слабость на 30 секунд

Rad-X (Профилактика)
- Таблетка, +50% сопротивления радиации на 10 минут
- Не снимает накопленную дозу!

Decontamination Shower
- Мультиблок 2x2x3
- Требует чистую воду
- Снимает дозу со скоростью 10 RAD/сек
- Работает только на игроков без брони

Radiation Scrubber
- Машина для очистки чанков
- Потребляет энергию и фильтры
- Скорость: 1 RAD/t за 1000 RF

9. РЕАКТОРЫ: 4 ТИРА

9.1 Tier 1: SMR (Small Modular Reactor)

Характеристики
- Размер: 5x5x5 (компактный)
- Тепловая мощность: 100-400 MWt
- Электрическая мощность: 30-150 MWe (через паровую турбину)
- Топливо: Low-Enriched Uranium (LEU, 3-5% U-235)

Конструкция

```
Слой 1-2: Бетонная подушка (радиационная защита)
Слой 3: Корпус реактора (Steel Casing) + ТВЭЛы
Слой 4: Теплообменник + Вода
Слой 5: Защитный купол (Concrete)
```

ТВЭЛ (Fuel Rod)
- Содержит UO₂ (диоксид урана)
- Срок службы: 20,000 тиков (в игре 16 часов)
- После использования: Spent Fuel (требует хранения!)

Системы безопасности
- Control Rods: Поглощают нейтроны (графит + бор)
- SCRAM: Аварийное отключение (2 секунды на полное опускание)
- Emergency Cooling: Пассивное охлаждение при отказе насосов

9.2 Tier 2: PWR (Pressurized Water Reactor)

Характеристики
- Размер: 9x9x12 мультиблок
- Мощность: 1,000-3,000 MWt / 300-1,000 MWe
- Давление: 155 бар (высокое давление!)
- Топливо: LEU или MOX (смесь U + Pu)

Первый контур (радиоактивный)
- Вода под давлением 155 бар
- Температура: 330°C (не кипит!)
- Проходит через активную зону

Второй контур (чистый)
- Парогенераторы обменивают тепло
- Образуется пар для турбин
- Никакой радиации!

Компоненты
- Steam Generator: 3x3x6, теплообменник
- Pressurizer: Поддерживает давление (электрический нагрев)
- Reactor Coolant Pump: Циркуляция воды (требует энергию!)
- Containment Vessel: Гермооболочка (при аварии — сдерживание)

9.3 Tier 3: RBMK (Канальный реактор большой мощности)

Характеристики
- Размер: 15x15x20 (огромный!)
- Мощность: До 4,000 MWt
- Топливо: LEU или Natural Uranium
- Замедлитель: Графит (не вода!)

Уникальная механика: Положительный паровой коэффициент
- При потере воды (образовании пара) реакция УСКОРЯЕТСЯ!
- Это реалистичная опасность Чернобыля
- Требует постоянного контроля

Конструкция
- Графитовые колонны: 1,600 штук в сетке
- Топливные каналы: В каждой колонне — 18 ТВЭЛов
- Система управления: 211 стержней (179 коротких, 32 длинных)
- Днище: Бетонная плита (при расплаве — кориумный источник)

Аварийные сценарии
1. Steam Explosion: При разгерметизации канала
2. Graphite Fire: При >700°C графит воспламеняется
3. Core Meltdown: Расплав активной зоны → Corium
4. Chernobyl Event: Полный взрыв реактора (загрязнение 50x50 чанков!)

9.4 Tier 4: Fusion Reactor (Токамак)

Характеристики
- Размер: 25x25x15 (огромное сооружение)
- Мощность: 500-5,000 MWe (зависит от топлива)
- Топливо: D-T (дейтерий-тритий), D-D, H-B (водород-бор)
- Температура плазмы: 150,000,000°C

Конструкция Токамака
- Toroidal Chamber: Вакуумная камера тора
- Magnetic Coils: Сверхпроводящие катушки (требуют -269°C!)
- Plasma Heater: Нагрев плазмы (RF + нейтральный пучок)
- Divertor: Удаление продуктов реакции (гелий)
- Blanket: Литиевый слой для бридинга трития

Топливные циклы

Цикл	Температура	Выход энергии	Сложность	
D-T	100 млн °C	1x	Базовая	
D-D	300 млн °C	0.8x	Средняя	
H-B (p-B11)	1 млрд °C	2x	Экстремальная (без нейтронов!)	

Системы поддержания
- Cryogenic Cooling: Жидкий гелий для катушек
- Vacuum System: Вакуумные насосы (10^-6 Паскалей)
- Tritium Breeding: Li-6 + нейтрон → Tritium
- Plasma Control: Алгоритмы стабилизации (можно программировать!)

10. ВСПОМОГАТЕЛЬНЫЕ СИСТЕМЫ

10.1 Управление и автоматизация

Reactor Control Computer
- Подключение к ComputerCraft / OpenComputers
- API для чтения температуры, мощности, управления стержнями
- Автоматические скрипты безопасности

Safety Logic Controller
- Hardcoded safety (нельзя отключить!)
- SCRAM при:
  - Температура > 110% номинальной
  - Потеря давления
  - Потеря питания систем управления
  - Сейсмическая активность (если есть мод на землетрясения)

10.2 Хранение отработанного топлива

Spent Fuel Pool
- Водяной бассейн 7x7x4
- Охлаждение отработанных ТВЭЛов (5 лет в игре)
- Требует циркуляцию воды!

Dry Cask Storage
- Контейнеры из бетона и стали
- После 5 лет в бассейне
- Пассивное воздушное охлаждение

Deep Geological Repository
- Подземное хранилище на глубине 50+ блоков
- Гранитная порода
- Вечное хранение (в игре)

10.3 Переработка отходов

Reprocessing Plant
- Разделение U и Pu из отработанного топлива
- MOX Fuel production
- Риск: Чистый плутоний = оружейный материал!

Waste Vitirification
- Заливка жидких отходов в стекло
- Безопасное долгосрочное хранение

🔧 ТЕХНИЧЕСКАЯ РЕАЛИЗАЦИЯ

Архитектура кода

Core Systems
1. RadiationCapability — ICapabilityProvider для чанков и игроков
2. MultiblockRegistry — система регистрации мультиблоков
3. FluidNetwork — собственная система труб (совместимость с Forge Fluids)
4. HeatSystem — тепловые сети (не RF, а °C и Джоули)
5. ChemistryRegistry — рецепты химических реакций с катализаторами

Ключевые классы

```java
// Радиация
IRadiationSource (interface) — для блоков/предметов
RadiationManager — глобальный менеджер распространения
PlayerRadiationCapability — накопленная доза

// Мультиблоки
AbstractMultiblockController — базовый класс
IMultiblockValidator — проверка структуры
MultiblockTank — мультиблок-жидкостное хранилище

// Химия
ChemicalReaction — рецепт с энтальпией, катализатором
Catalyst — модификатор скорости реакции
ReactionChamber — базовый класс химреакторов
 ```
Конфигурация:

 ```java
 radiation:
  enabled: true
  spread_rate: 0.01  # 1% в тик
  decay_rate: 0.001  # полураспад чанка
  player_death_threshold: 1000  # RAD
  
reactors:
  meltdown_enabled: true
  explosion_power_multiplier: 1.0
  rbmk_positive_void_coefficient: true  # Хардкор!
  
performance:
  max_multiblock_size: 1000  # блоков
  fluid_tick_rate: 20  # раз в секунду
  ```
